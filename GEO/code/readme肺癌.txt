1.训练集gse19188
GSE19188.csv为表达谱
GSE19188_clinic.csv为样本信息

2.根据label把样本分为疾病和正常
对疾病和正常找差异基因(使用Rlimma包)
筛选选取FC差值2倍且p.adjust<0.05的基因,共1483个(-deg.csv)
筛选P值在top100的基因做聚类分析(符合条件的基因聚类.png)

3. 在差异基因中筛选和预后相关的基因
在1483个候选基因中根据单因素cox回归分析,筛选ph>0.05 cox<0.05的基因
和生存显著相关的基因有44个(-cox-gene.csv)

4. 配成基因对根据二项分布检验筛选差异基因对
识别27个基因对(-gene-pair.csv)

5. 根据基因对打分在癌症样本中识别高低风险
由于特征识别是在癌和正常中识别,但是分类时是在全癌样本中分类,因此设置70%的投票阈值(9对)
若基因对打分超过9则认为是低风险样本,否则为高风险

6. 对高低风险识别差异基因、聚类、富集
NORMAL-TUMOR_kegg_up_down 为KEGG通路富集结果

NORMAL-TUMOR_dotplot_gene_diff_ + BP、CC、MF
这三个是不同的功能富集结果，效果都一样，自己选一个和老师展示就行
